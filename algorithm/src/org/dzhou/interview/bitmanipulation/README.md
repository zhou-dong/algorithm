# Bit Manipulation

```java
0110 + 0010 = 1000, 0011 * 0101 = 1111, 0110 + 0110 = 1100
```

```java
0011 + 0010 = 0101, 0011 * 0011 = 1001, 0100 * 0011 = 1100
```

```java
0110 - 0011 = 0011, 1101 >> 2 = 0011, 1101 ^ (~1101) = 1111
```

```java
1000 - 0110 = 0010, 1101 ^ 0101 = 1000, 1011 & (~0 << 2 ) = 1000
```
